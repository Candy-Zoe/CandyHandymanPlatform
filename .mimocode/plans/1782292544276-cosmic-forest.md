# 万能工匠平台 (Universal Handyman Platform) - 实现计划

## 项目概述
一个连接技能工匠与顾客的O2O平台，支持工匠展示技能、发布服务，顾客浏览下单、在线沟通。包含后端API、Android手机端、WPF电脑端、微信小程序端。

## 技术栈
| 层级 | 技术 |
|------|------|
| 后端 | .NET 8 Web API + Entity Framework Core + SQLite |
| Android | Kotlin + Jetpack Compose + Hilt + Material 3 |
| 桌面端 | WPF (.NET 8) + MVVM + CommunityToolkit.Mvvm |
| 微信小程序 | 原生 WXML/WXSS/JS |

### 1.4 API端点设计

**认证模块 /api/auth/**
- POST /register - 注册
- POST /login - 登录（返回JWT+RefreshToken）
- POST /refresh - 刷新令牌
- POST /forgot-password - 找回密码

**用户模块 /api/users/**
- GET /me - 获取当前用户信息
- PUT /me - 更新用户信息
- PUT /me/avatar - 上传头像
- GET /{id} - 获取指定用户信息

**工匠模块 /api/handymen/**
- GET / - 工匠列表（支持筛选：分类、评分、距离、关键词）
- GET /{id} - 工匠详情
- POST /profile - 创建/更新工匠档案
- PUT /availability - 更新接单状态
- GET /my-services - 我发布的服务

**服务模块 /api/services/**
- GET / - 服务列表（支持分类筛选、搜索、排序）
- GET /{id} - 服务详情（含媒体）
- POST / - 发布服务
- PUT /{id} - 更新服务
- DELETE /{id} - 删除服务
- POST /{id}/media - 上传服务媒体（图片/视频）
- DELETE /{mediaId}/media - 删除服务媒体

**订单模块 /api/orders/**
- POST / - 创建订单
- GET / - 订单列表（按角色：我的订单/我接的单）
- GET /{id} - 订单详情
- PUT /{id}/accept - 接单
- PUT /{id}/start - 开始服务
- PUT /{id}/complete - 完成服务
- PUT /{id}/cancel - 取消订单
- PUT /{id}/dispute - 申请争议

**聊天模块 /api/chat/**
- GET /conversations - 会话列表
- GET /conversations/{id}/messages - 消息记录
- POST /messages - 发送消息
- PUT /messages/read - 标记已读

**评价模块 /api/reviews/**
- POST / - 提交评价
- GET /handyman/{handymanId} - 工匠收到的评价
- GET /my-given - 我发出的评价

**分类模块 /api/categories/**
- GET / - 获取所有分类（树形结构）
- GET /{id}/services - 某分类下的服务

**支付模块 /api/payments/**
- POST /create - 创建支付
- GET /history - 支付记录

**通知模块 /api/notifications/**
- GET / - 通知列表
- PUT /{id}/read - 标记已读
- PUT /read-all - 全部已读

- **聊天**: SignalR实时通信 / WebSocket
- **地图**: 高德/百度地图SDK获取位置
- **推送**: Firebase Cloud Messaging (FCM) / 华为HMS
